# CS330
For the Halloween scene, I basically approached the whole thing by breaking it down into small pieces I could actually manage. Instead of thinking “build a whole scene,” I focused on one object at a time. the pumpkin, the table, the lantern, the candy, the ghost and figured out how to shape each one using simple geometry. Working this way helped me get better at designing objects that fit together visually, and it made the whole project feel less overwhelming.
My process was pretty much trial and error. I’d build something, drop it into the scene, check the lighting, fix whatever looked weird, and then move on. Lighting was a big learning moment for me tiny changes in ambient or specular values completely changed the mood, so I spent a lot of time tweaking that. Each milestone pushed me to clean up my code, fix textures, and make the objects look more intentional.
Overall, this project showed me how useful graphics work can be. The skills translate into anything visual game dev, UI/UX, simulations, or even just communicating ideas in a clearer way. It definitely gave me a better sense of how to build something step by step and make it look the way I imagined.
